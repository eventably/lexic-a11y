  • Ctrl + B - Bold
  • Ctrl + I - Italic
  • Ctrl + U - Underline
  • Ctrl + K - Insert Link
  • Ctrl + Shift + 7 - Numbered List
  • Ctrl + Shift + 8 - Bullet List
  • Ctrl + Alt + 1-6 - Headings 1-6
